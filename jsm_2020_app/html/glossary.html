• Affiliation - A general category describing the nature of the ASA president's employment at the time they were president. Values include Academia, Government, Industry, and Unknown.
  • Average Sentence Length - The average number of words in sentences within a document (speech).
  • Average Word Length - The average number of characters in words within a document (speech).
  • Flesch-Kincaid Age Score- This is a transformation that translates the Flesh-Kincaid Grade Score into the age, in years, that one would need to be to fully understand the writing. For more information see the koRpus documentation.
  • Flesch-Kincaid Grade Score - A measure of the readability of a document (speech) that presents a score in the form of a U.S. grade level that one would need to be at to fully understand the writing. The score is calculated using Average Sentence Length and the average number of syllables per word. For more information see the koRpus documentation.
  • Length Rank - The rank order of the speech in terms of Number of Words with lower numbers meaning longer speeches and higher numbers meaning shorter speeches.
  • Named Entity Recognition - The use of a model to identify chunks of text that are important in some way. In the case of this app, we used the spaCy library in Python to extract names and groups.
  • Number of Words - The total number of running word forms (tokens) in a document (speech).
  • Number of Unique Words - The total number of distinct word forms (types) in a document (speech).
  • Sentiment - Approximate polarity (extremity of positivity or negativity) expressed in a document (speech or sentence) as determined by assigning words a score from a sentiment dictionary and aggregating those scores accounting for negations. For more information see the qdap documentation for polarity score.
  • Topic Models - For more information see the topicmodels documentation for LDA.
    • Word Loadings (Beta) - Logarithmized parameters of the word distribution for each topic.
    • Topic Loadings (Gamma) - Parameters of the posterior topic distribution for each document. Documents can have non-zero loadings for multiple topics. The Gamma values for a document sum to 1.
    • Main Topic - The ID of the topic with the highest Topic Loading (Gamma) for a document.
  • Type-Token Ratio (TTR) - The number of distinct words (types) divided by the total number of words (tokens) in a document. This is often used as a measure of complexity with higher values meaning one would have to parse and understand a greater proportion of unique words. For more information see the koRpus documentation for TTR.
